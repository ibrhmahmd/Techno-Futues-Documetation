What are CSS Preprocessors?

CSS preprocessors extend CSS with features like variables, mixins, and nesting, making your stylesheets more maintainable and easier to write.

SASS vs LESS

Variables

SASS

$primary-color: #3498db;

LESS

@primary-color: #9b59b6;

Nesting

SASS

.button {
  &-primary { /* styles */ }
  &:hover { /* styles */ }
}

LESS

.button {
  &-primary { /* styles */ }
  &:hover { /* styles */ }
}

Mixins

SASS

@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

.selector { @include flex-center; }

LESS

.flex-center() {
  display: flex;
  justify-content: center;
  align-items: center;
}

.selector { .flex-center(); }

Extend/Inheritance

SASS

%message {
  padding: 10px;
  border: 1px solid #ccc;
}

.success { @extend %message; }

LESS

.message {
  padding: 10px;
  border: 1px solid #ccc;
}

.success { &:extend(.message); }

Loops

SASS

@for $i from 1 through 5 {
  .mt-#{$i} { 
    margin-top: #{$i * 0.5}rem; 
  }
}

LESS

.generate-margins(@n, @i: 1) when (@i =< @n) {
  .mt-@{i} { 
    margin-top: (@i * 0.5rem); 
  }
  .generate-margins(@n, (@i + 1));
}
.generate-margins(5);

Functions

SASS

@function em($pixels, $context: 16) {
  @return #{$pixels/$context}em;
}

LESS

.em(@pixels, @context: 16) {
  @return: unit(@pixels / @context, em);
}

Maps

SASS

$colors: (
  primary: #3498db,
  secondary: #2ecc71
);

@each $name, $color in $colors {
  .text-#{$name} { color: $color; }
}

LESS

@colors: {
  primary: #9b59b6;
  secondary: #e67e22;
}

each(@colors, {
  .text-@{key} { color: @value; }
});

Conditionals

SASS

@mixin theme-colors($theme) {
  @if $theme == "light" {
    background: white;
  } @else if $theme == "dark" {
    background: black;
  } @else {
    background: gray;
  }
}

LESS

.theme-colors(@theme) when (@theme = light) {
  background: white;
}

.theme-colors(@theme) when (@theme = dark) {
  background: black;
}

.theme-colors(@theme) when (default()) {
  background: gray;
}

How to compile:

Install SASS:
npm install -g sass
Compile to CSS:
sass input.scss output.css
Watch for changes:
sass --watch input.scss:output.css
Install LESS:
npm install -g less
Compile to CSS:
lessc input.less output.css
Watch for changes (requires additional tool like nodemon):
npm install -g nodemon
nodemon --exec "lessc input.less output.css" --watch input.less
