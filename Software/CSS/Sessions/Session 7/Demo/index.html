CSS Transitions

Transitions enable you to define the transition between two states of an element.

Hover Me!

Generated CSS

.transition-box {
  transition: all 0.3s ease;
}

.transition-box:hover {
  transform: scale(1.1);
  background: #4f46e5;
  color: white;
}

CSS Animations

Create complex animations using @keyframes and control them with animation properties.

Animate Me!

Generated CSS

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

.animation-box {
  animation: bounce 1s infinite;
}

Performance & Best Practices

Learn which properties are best for smooth animations and how to optimize performance.

Good for Performance

transform & opacity

Uses GPU acceleration for smooth animations.

Bad for Performance

width, height, margin

Causes layout recalculations and repaints.

Respect User Preferences

Always respect the user's motion preferences:

@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}
