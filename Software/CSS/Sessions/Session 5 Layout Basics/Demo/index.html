Floats

Float Left

This text flows around the floated element. Floats were traditionally used for layouts but are now mainly used for text wrapping around images and other content.

Float Right

This paragraph has the clearfix class to clear the floats. Notice how it appears below both floated elements.

CSS Code:

.float-left { float: left; margin-right: 20px; }
.float-right { float: right; margin-left: 20px; }
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

}

/* Shadow Effect */
.shadow .hover-demo:hover {
    box-shadow: 0 10px 20px rgba(0,0,0,0.2);
    transform: translateY(-3px);
    transition: all 0.3s ease;
}

/* Transform */
.transform .hover-demo:hover {
    transform: rotate(5deg) translateX(10px);
    transition: transform 0.3s ease;
