Media Queries

Adjust the browser width to see how the layout changes at different breakpoints.

Mobile (< 576px)
Tablet (≥ 576px)
Laptop (≥ 992px)
Desktop (≥ 1200px)

CSS Code Example:

/* Mobile-first approach */
.element {
  /* Base styles for mobile */
  padding: 1rem;
  font-size: 1rem;
}

/* Tablet and up */
@media (min-width: 576px) {
  .element {
    padding: 1.5rem;
    font-size: 1.1rem;
  }
}

/* Desktop */
@media (min-width: 992px) {
  .element {
    padding: 2rem;
    font-size: 1.25rem;
  }
}

Flexible Units

These units adjust based on viewport size or parent element.

10vw (10% of viewport width)
10vh (10% of viewport height)
2rem (relative to root font-size)
2em (relative to parent font-size)

Fluid Typography

This text size adjusts smoothly between a minimum and maximum size based on viewport width.

1024px 
/* Fluid typography with clamp() */
h1 {
  font-size: clamp(1.5rem, 4vw + 1rem, 3rem);
  /* Minimum: 1.5rem, Preferred: 4vw + 1rem, Maximum: 3rem */
}

  <source media="(min-width: 768px)" srcset="medium.jpg">
  <img src="small.jpg" alt="Description" style="width: 100%;">

Responsive Layout Patterns

CSS Grid for Responsive Layouts

.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
  padding: 1rem;
}

/* Responsive grid with media queries */
@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: repeat(3, 1fr);
  }
}
